## Get Started

To play Miners Online, follow these steps:

import { Step, Steps } from 'fumadocs-ui/components/steps';

<Steps>
<Step>
### Download Minecraft: Java Edition

Make sure you have a valid copy of Minecraft: Java Edition installed on your computer. You can purchase it from the [official Minecraft website](https://minecraft.net).
</Step>
<Step>
### Launch Minecraft and Add Server

1. Open the Minecraft Launcher and log in with your account.
2. Start the game and navigate to the "Multiplayer" section.
3. Click on "Add Server" and enter the server address: `play.minersonline.uk`
4. Click "Done" to save the server to your list.
</Step>
<Step>
### Join the Server
Select "Miners Online" from your server list and click "Join Server" *(or just double click)* to connect and start playing!
</Step>
</Steps>
